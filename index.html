Quiz Rules

The aim of the game is to answer as many of the questions correctly to earn the highest score. The game will end once all of the questions have been answered.

How To Play:

  • Press Play to start the quiz game
  • Once the game starts, you will be presented with a question, select one of the 4 options to answer the question
  • If you answer correctly, the selected answer will turn green, if answered incorrectly this will turn red
  • Once the question is answered, you will be presented with the next question, same steps as above apply
  • You can end the game anytime by selecting End Quiz (Warning: Progress will not be saved)
  • Once all the questions have been answered, the quiz game will end
  • You will be presented with the option to enter a name to record the score earned during this game
  • Click Play Again to start the quiz game again
  • Click Close to go back to the home screen
  • If you wish to view the scores table, then from the Home section, select Highscores
